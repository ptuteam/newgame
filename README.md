## Build Instructions

1. Install grunt

2. Clone repository:

   ```bash 
   git clone --recursive https://github.com/ptuteam/quiz.git
   ```
   
   or
   
   ```bash 
   git clone https://github.com/ptuteam/quiz.git
   git submodule update --init
   ```

3. Install npm dependencies:

   ``` npm install ```

4. Build a server:

   ``` grunt build ```
5. Run:

   ``` grunt  ```
